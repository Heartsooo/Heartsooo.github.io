# 背部解剖速览  

![背部肌肉示意](https://upload.wikimedia.org/wikipedia/commons/3/39/Back_muscles_labeled.svg)

## 1 · 主要肌群  

| 肌群 | 位置概览 | 核心功能 | ExRx 资料 |
| :-- | :-- | :-- | :-- |
| **背阔肌**<br>(Latissimus dorsi) | 覆盖中-下背外侧，呈扇形展开 | 肩关节伸展、内收、内旋；协助下拉躯干与呼吸 | <https://exrx.net/Muscles/LatissimusDorsi> |
| **斜方肌**<br>(Trapezius，上／中／下束) | 枕骨 → T12，三角形覆盖上背 | 上束耸肩与颈伸展；中束肩胛后收；下束肩胛下压、外旋 | <https://exrx.net/Muscles/Trapezius> |
| **菱形肌**<br>(Rhomboid major / minor) | 位于斜方肌深层，胸椎 → 肩胛内缘 | 肩胛骨内收、下旋，维持上背姿势 | <https://exrx.net/Muscles/Rhomboids> |
| **竖脊肌**<br>(Erector spinae) | 沿脊柱两侧纵向排列 | 脊柱伸展、侧屈；维持站立与躯干稳定 | <https://exrx.net/Muscles/ErectorSpinae> |
| **大 / 小圆肌**<br>(Teres major / minor) | 肩胛外侧缘 | 大圆肌：肩内收、内旋；小圆肌：肩外旋，属旋转袖 | <https://exrx.net/Muscles/TeresMajor> · <https://exrx.net/Muscles/TeresMinor> |
| **肩袖后群**<br>(Infraspinatus 等) | 肩胛后侧 | 肩外旋并稳定肱骨头，防止撞击 | <https://exrx.net/Muscles/Infraspinatus> |
| **肩胛提肌**<br>(Levator scapulae) | 颈椎 → 肩胛上角 | 提肩胛骨，协助颈侧屈和旋转 | <https://exrx.net/Muscles/LevatorScapulae> |

## 2 · 关键关节（Articulations）

| 关节 / 功能性接触面 | 类型 | 主要运动 & 训练相关性 | ExRx 资料 |
| :-- | :-- | :-- | :-- |
| **椎间关节**<br>(Intervertebral & Facet) | 椎体间纤维软骨 + 小关节面 | 脊柱屈伸、侧屈、旋转；竖脊肌与深层稳定肌控制 | <https://exrx.net/Articulations/Vertebral> |
| **胸椎-肋关节**<br>(Costovertebral) | 滑车-平面 | 呼吸时肋骨抬升/下沉；硬拉、划船需保持胸椎伸展 | <https://exrx.net/Articulations/Costovertebral> |
| **腰骶关节**<br>(Lumbosacral) | 椎间盘 + 滑车 | 连接躯干与骨盆，负重转移；硬拉、深蹲中核心稳固关键 | <https://exrx.net/Articulations/LumboSacral> |
| **肩胛-胸壁**<br>(Scapulothoracic) | 功能性滑动面 | 肩胛上/下旋、前伸、后缩；影响引体向上、划船效率 | <https://exrx.net/Articulations/ScapuloThoracic> |
| **肩锁关节**<br>(Acromioclavicular) | 平面滑动 | 微小调节肩胛动作，维持锁骨-肩峰间角度 | <https://exrx.net/Articulations/AcromioClavicular> |
| **胸锁关节**<br>(Sternoclavicular) | 鞍状-球窝 | 唯一连接上肢与躯干的真关节；肩带整体活动范围“枢纽” | <https://exrx.net/Articulations/SternoClavicular> |
| **肱肩关节**<br>(Glenohumeral) | 球窝滑液关节 | 多轴运动（屈伸、内外旋、外展等）；背部训练主要动力枢纽 | <https://exrx.net/Articulations/ShoulderGlenohumeral> |

> **提示**  
> 充分了解肌肉-关节结构后，再去安排热身 → 训练 → 拉伸，可显著提升效率并降低伤害风险。
