| 肌群 | 位置概览 | 核心功能 | ExRx 资料 |
| :-- | :-- | :-- | :-- |
| **背阔肌**<br>(Latissimus dorsi) | 覆盖中-下背外侧，呈扇形展开 | 肩关节伸展、内收、内旋；协助下拉躯干与呼吸 | <https://exrx.net/Muscles/LatissimusDorsi> |
| **斜方肌（上束）**<br>(Trapezius Upper) | 枕骨 → 锁骨外侧 | 耸肩、颈伸展、头部侧屈 | <https://exrx.net/Muscles/TrapeziusUpper> |
| **斜方肌（中束）**<br>(Trapezius Middle) | C7-T3 → 肩峰、肩胛脊 | 肩胛骨内收、固定 | <https://exrx.net/Muscles/TrapeziusMiddle> |
| **斜方肌（下束）**<br>(Trapezius Lower) | T4-T12 → 肩胛脊下部 | 肩胛骨下压、上旋、固定 | <https://exrx.net/Muscles/TrapeziusLower> |
| **菱形肌**<br>(Rhomboid major / minor) | 位于斜方肌深层，胸椎 → 肩胛内缘 | 肩胛骨内收、下旋，维持上背姿势 | <https://exrx.net/Muscles/Rhomboids> |
| **竖脊肌**<br>(Erector spinae) | 沿脊柱两侧纵向排列 | 脊柱伸展、侧屈；维持站立与躯干稳定 | <https://exrx.net/Muscles/ErectorSpinae> |
| **大 / 小圆肌**<br>(Teres major / minor) | 肩胛外侧缘 | 大圆肌：肩内收、内旋；小圆肌：肩外旋，属旋转袖 | <https://exrx.net/Muscles/TeresMajor> · <https://exrx.net/Muscles/TeresMinor> |
| **肩袖后群**<br>(Infraspinatus 等) | 肩胛后侧 | 肩外旋并稳定肱骨头，防止撞击 | <https://exrx.net/Muscles/Infraspinatus> |
| **肩胛提肌**<br>(Levator scapulae) | 颈椎 → 肩胛上角 | 提肩胛骨，协助颈侧屈和旋转 | <https://exrx.net/Muscles/LevatorScapulae> |

## 2 · 关键关节（带图）

### 椎间关节（Intervertebral）

### 肩锁关节（Acromioclavicular）

- 功能：微调肩胛动作，维持锁骨-肩峰角度

### 胸锁关节（Sternoclavicular）
