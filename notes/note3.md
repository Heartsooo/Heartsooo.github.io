## ⚔️ 进攻通用建议 {#进攻通用建议}

1. **双闪破点**：左键直抛给近角；0.2 s 后右键反弹给深处，避免被反打。  
2. **刀-闪二连**：利用**刀落地 0.45 s 声效**转移注意力，在同方向补闪击穿。  
3. **FRAG 拆常规高台 / 夹角**：Icebox Nest、Ascent Heaven、Split Heaven 等必练。  
4. **大招 NULL/cmd**：可打“**1-3-1**”或“**4-1 断后**”；进站后队友贴身站复活圈周围提供枪线保护。

## 🛡️ 防守通用建议 {#防守通用建议}

| 场景 | 推荐做法 | 备注 |
| --- | --- | --- |
| **前线对枪** | 预留刀；确认多人 Rush 才拉 EQ 闪拉距离 | 刀 CD 45 s，别首轮就交 |
| **拉锯补位** | 刀封 choke + FRAG 拖节奏；让队友烟封关键角 | 典型 Haven A / Icebox B |
| **残局 1 v X** | 手握 FRAG > 刀；先斩 Default → 逼拆 | 拆包声 3 s 起炸，注意节奏 |
| **NULL/cmd 反 retake** | 敌有 Sova Drone / Fade Prowler 时先开 | 抑制 info 源头；反制 Raze |
